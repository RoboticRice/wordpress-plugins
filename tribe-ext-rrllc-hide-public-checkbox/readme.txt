Event Tickets' RSVP ticket Public Attendance Checkbox will be hidden.

== Description ==

This plugin hides the checkbox for the RSVP tickets from view. No longer will your customers be forced to see "Hide my attendee data from public view." Developed by Robotic Rice LLC and not affiliated with Event Tickets. Currently does nothing.
